# MIPDB_Toolkit
## MIPDB: A High-Quality Keypoint Annotation Dataset and Benchmark for Maize in Complex Field Scenes
This repository contains the code used for the keypoint detection and semantic segmentation in the paper "MIPDB: A High-Quality Keypoint Annotation Dataset and Benchmark for Maize in Complex Field Scenes" by Jianye Chang, Panpan Wang, Wenpeng Deng, Yuchen Pan, Mengtao Wang, Zhihao Hou, Jason John Walsh, Eleni Mangina, Sonia Negrão, Yun Zhou, Zhen Zhang, Hailin Liu, Jue Ruan.
